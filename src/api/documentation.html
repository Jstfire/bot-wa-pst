WhatsApp API Documentation

BPS Buton Selatan - Messaging API for sending WhatsApp messages

Introduction

This API allows you to send WhatsApp messages programmatically using the BPS Buton Selatan WhatsApp Bot infrastructure. It provides endpoints for sending text messages and images to WhatsApp numbers.

Authentication

All API requests require authentication using a token-based approach. The token must be included in the HTTP header of each request.

Important: Keep your API tokens secure. Do not expose them in client-side code or public repositories.

Authorization Header

Authorization: Bearer YOUR_API_TOKEN

Token Management

Tokens are managed by administrators through the admin interface or API endpoints. Each token has a unique identifier and can be revoked if needed.

API Endpoints

POST /api/send-message

Send a text message to a WhatsApp number

Request Headers

Name Description
Content-Type application/json
Authorization Bearer YOUR_API_TOKEN

Request Body Parameters

Parameter Type Description
number (required) string The WhatsApp number to send to. Can be with or without country code.
Example: "085123456789" or "6285123456789"
message (required) string The message text to send to the recipient

Example Request

{
  "number": "085123456789", 
  "message": "Hello from the API!"
}

Example Response

{
  "success": true,
  "message": "Message sent successfully",
  "to": "6285123456789"
}

POST /api/send-image

Send an image with optional caption to a WhatsApp number

Request Headers

Name Description
Content-Type application/json
Authorization Bearer YOUR_API_TOKEN

Request Body Parameters

Parameter Type Description
number (required) string The WhatsApp number to send to. Can be with or without country code.
Example: "085123456789" or "6285123456789"
imageUrl (required) string URL to the image to send. Must be a publicly accessible URL
caption (optional) string Optional caption text for the image

Example Request

{
  "number": "085123456789",
  "imageUrl": "https://example.com/image.jpg",
  "caption": "Check out this image!"
}

Example Response

{
  "success": true,
  "message": "Image message sent successfully",
  "to": "6285123456789"
}

POST /api/check-number

Check if a phone number exists on WhatsApp before sending messages

Request Headers

Name Description
Content-Type application/json
Authorization Bearer YOUR_API_TOKEN

Request Body Parameters

Parameter Type Description
number (required) string The WhatsApp number to check. Can be with or without country code.
Example: "085123456789" or "6285123456789"

Example Request

{
  "number": "085123456789"
}

Example Response

{
  "success": true,
  "exists": true,
  "number": "6285123456789"
}

Error Responses

The API uses standard HTTP status codes to indicate the success or failure of a request.

Status Code Description
200 OK Request successful
400 Bad Request Invalid request parameters
401 Unauthorized Missing or invalid API token
403 Forbidden Not allowed to access this resource
404 Not Found Resource not found or phone number does not exist on WhatsApp
500 Internal Server Error Server error while processing the request
503 Service Unavailable WhatsApp service is not available

Error Response Format

{
  "success": false,
  "message": "Error description",
  "error": "Detailed error information (optional)"
}

Integration Examples

JavaScript/Node.js Example

const fetch = require('node-fetch');

async function sendWhatsAppMessage(number, message) {
  try {
    const response = await fetch('http://your-api-server.com/api/send-message', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_TOKEN'
      },
      body: JSON.stringify({
        number: number,
        message: message
      })
    });
    
    const result = await response.json();
    console.log('Message sent:', result);
    return result;
  } catch (error) {
    console.error('Error sending message:', error);
    throw error;
  }
}

// Example usage
sendWhatsAppMessage('085123456789', 'Hello from Node.js!');

PHP Example

<?php
function sendWhatsAppMessage($number, $message) {
  $url = 'http://your-api-server.com/api/send-message';
  
  $data = array(
    'number' => $number,
    'message' => $message
  );
  
  $options = array(
    'http' => array(
      'header'  => "Content-type: application/json\r\n" .
                   "Authorization: Bearer YOUR_API_TOKEN\r\n",
      'method'  => 'POST',
      'content' => json_encode($data)
    )
  );
  
  $context  = stream_context_create($options);
  $result = file_get_contents($url, false, $context);
  
  if ($result === FALSE) {
    return array('success' => false, 'message' => 'Error sending message');
  }
  
  return json_decode($result, true);
}

// Example usage
$result = sendWhatsAppMessage('085123456789', 'Hello from PHP!');
print_r($result);
?>

Python Example

import requests
import json

def send_whatsapp_message(number, message):
    url = 'http://your-api-server.com/api/send-message'
    
    headers = {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_TOKEN'
    }
    
    data = {
        'number': number,
        'message': message
    }
    
    response = requests.post(url, headers=headers, data=json.dumps(data))
    
    return response.json()

# Example usage
result = send_whatsapp_message('085123456789', 'Hello from Python!')
print(result)
